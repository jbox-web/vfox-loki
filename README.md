# vfox-loki

[![CI](https://github.com/jbox-web/vfox-loki/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-loki/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [loki](https://github.com/grafana/loki)

## Install

```
mise plugin install loki https://github.com/jbox-web/vfox-loki.git
```
